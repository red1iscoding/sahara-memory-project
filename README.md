This is the sahara memory project. its a project by redouane to undrestand the dessertification in the sahara dessert

Phase 1 — Data exploration: Google Earth Engine, Sentinel-2/MODIS NDVI data for Algeria, EDA\
Phase 2 — ETL pipeline: Airflow, PostgreSQL, automated satellite data ingestion \
Phase 3 — ML model: land cover classifier (Random Forest → CNN), time-series trend detection, SHAP explainability\
Phase 4 — Heritage risk layer: EAMENA database, spatial joins, risk scoring per archaeological site\
Phase 5 — Deployment: Streamlit dashboard, FastAPI, Docker, cloud deployment\
